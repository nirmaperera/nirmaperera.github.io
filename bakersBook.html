Baker's Book

A digital recipe book for enthustatic bakers

Tech Stack

  • React
  • Redux
  • Firebase
  • View Code
  • View Wireframes

My Role

Frontend developer and designer

During the beginning of the project, I designed the mockups and chose the color palette. Some of the features I built was posting a recipe, which was done by using a form, searching for a recipe by filtering through the name of the recipe, landing page/responsive navbar, and profile components.

Project Difficulties

The original idea of the application was to build a fun and interactive way to post a recipe using animation, etc. A example of our inspiration wasÂ Poke Bar Menu. Scaling down the project was something we had trouble with, because as a team we had to decide which features were the most important. In addition, learning redux for the first time came with some difficulty.

Project Solutions

Scaling down the project made the application way easier to do given the time constraints. As a team, we decided to build out the most important features of the application. For example, a simple form with some animation using animate.css. Depending if we had time, we would build a more interactive form. To learn redux, tutorials on youtube and other online resources helped to understand the javascript library.

Application Features

  • Registration
  • Login
  • User authentication
  • Profile (a list of user made recipes)
  • Search for a recipe by category
  • Post a recipe
  • View a recipe

What's missing?

Responsiveness throughout the whole application and adding a comments/rating feature.
