Telechat

Real time chat application

  • View Code

Full Stack Application

This project is a real time chat application where users can join a room with their friends and send emojis, images, stickers, gifs, and text. In addition for this application, I focused on creating beautiful UI by using css animations, shadows, and choosing a consistent color palette.

Project Difficulties

The main difficulty for this project was understanding socket.io and how to integrate it with react. Specifically how to send messages to the chat room. In addition, sending other forms of messages such as gifs and images from users device also was a challenge.

Project Solutions

Socket.io “is a library that enables real-time, bidirectional and event-based communication between the browser and the server”. Io.on is used for incoming socket connections and once there is a new connection, then we’ll be attaching different event listeners to the stream. For example, socket.emit(‘sendMessage’) is used to send messages from one client to another client(user) and socket.emit(‘disconnect’) is called when a user quits the chat room.
To send gifs, I decided to use the giphy API along side react-giphy-searchbox package to make a http request to the search endpoint. GIPHY Search gives you instant access to our library of millions of GIFs and Stickers by entering a word or phrase.  Similarly, for sending images from a user device I used the FileReader(). I made sure to have accepted field in input to only accept images, (accept=“image/*).
Some other features that the chat app is No duplicate users can join the same room which means no two users with the same name can join the same room. Also the app checks if there is connection error, and if there is, the socket is disconnected and the user redirected to the join page.

Application Features

  • login/join chat room
  • send stickers or gifs from giphy api
  • Send only image files from device
  • Send emoji

What's Missing?

Some features that would be beneficial for this build would be to implement reactions to a message (similar to imessage), and detect when a user is typing.
